🔐 Autenticación requerida

El error 401 Unauthorized ocurre porque los navegadores modernos bloquean las cookies de sesión entre dominios distintos (github.io → qlikcloud.com). Tienes dos opciones para resolverlo:

🪟 Opción A — Ventana de login RECOMENDADA

Abre Qlik Cloud en una ventana emergente, inicia sesión, y luego carga las gráficas.

1
Haz clic en "Abrir login de Qlik"
2
Inicia sesión en la ventana que aparece
3
Cierra esa ventana y pulsa "Recargar gráficas"

🗝️ Opción B — API Key

Si tienes una API Key de Qlik Cloud, pégala aquí para autenticarte directamente.

1
Qlik Cloud → perfil → API Keys → Create new key
2
Copia la key generada (solo se muestra una vez)
3
Pégala aquí y haz clic en Conectar
